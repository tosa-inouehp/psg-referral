# PSG検査入院 受け入れ案内サイト

高知脳神経外科病院からの紹介患者（簡易SAS検査でSAS疑い）を、井上病院でPSG入院検査として受け入れるための案内ページ群です。

## GitHub Pages URL

**https://tosa-inouehp.github.io/psg-referral/**

---

## ファイル構成

```
docs/                        # GitHub Pages 公開ディレクトリ
├── index.html               # 受け入れ案内トップ（フロー図・入院書類リンク含む）
├── checklist.html           # 予約・入院前・入院当日・退院時チェックリスト
├── contact.html             # 担当窓口・連絡先
├── admission-form.html      # 入院申込書（兼誓約書）・緊急連絡先表（印刷用）
├── admission-plan.html      # 入院診療計画書 兼 入院同意書（印刷用）
└── assets/
    ├── style.css            # 共通スタイル（CSS変数・レスポンシブ・印刷用）
    └── flow.svg             # 予約〜退院フロー図（8ステップ）
参考資料/                     # 当院既存PSG入院資料（制作参考用）
README.md
```

## ページ内容

### index.html（受け入れ案内トップ）
- PSG検査入院の概要・注意事項
- 入院時刻：原則 **15:30まで**（16:00よりPSG機器装着開始）
- PSG機器装着：検査技師・フクダライフテック技師が実施
- 機器装着後の食事・飲水（ストロー使用）可
- 会計は**後日会計**
- 入院書類（admission-form・admission-plan）へのリンクボタン
- 予約〜退院のフロー図（SVG）

### checklist.html（チェックリスト）
- 4フェーズ：予約時 / 入院前 / 入院当日 / 退院時
- JSによる進捗バー・リセットボタン

### contact.html（連絡先・窓口）
- **予約受付・書類手配**：相談室　TEL 088-852-2131 / FAX **088-852-2133** / 平日 9:00〜17:00
- **入院当日・病棟対応**：井上病院 外来・医療病棟　TEL 088-852-2131 / 9:00〜17:00
- **機器手配**：フクダライフテック（井上病院側で直接手配・紹介元からの連絡不要）
- 予約依頼票FAX送付手順

### admission-form.html（入院申込書・緊急連絡先表）
- **1ページ目**：入院申込書（兼誓約書）― 入院者・申込者・連帯保証人の各記入欄
- **2ページ目**：緊急連絡先表（5名分）・緊急時署名欄
- A4・印刷専用レイアウト（@media print）
- 患者様が記入のうえ**入院当日持参**

### admission-plan.html（入院診療計画書 兼 入院同意書）
- 1ページ・A4印刷用
- 説明医師・看護師記入欄、病名（睡眠時無呼吸症候群）・病棟（医療病棟2F）
- 外来/入院日/退院日の3列構成で経過説明を記載
- 食事・安静度・排泄・内服薬の案内（当院PSG方針に準拠）
- 患者同意署名欄（令和 年 月 日）
- **高知脳神経外科病院**で印刷・患者説明・署名取得のうえ入院当日持参

## GitHub Pages の設定方法

1. リポジトリを GitHub にプッシュ
2. リポジトリの **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / Folder: `/docs`
5. Save → 数分後に公開URL発行

## 印刷時の注意

ブラウザの印刷ダイアログで **用紙サイズ「A4」** を必ず選択してください。  
CSSで `@page { size: A4 portrait; }` を指定していますが、ブラウザ・プリンター設定によって上書きされる場合があります。
